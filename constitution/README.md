# Constitution

This directory contains the foundational documents of Admissibility Theory.

These three documents are the minimum required to understand the framework:

| Document | Answers |
|---|---|
| `four_axioms.md` | What conditions must a claim satisfy to be admissible? |
| `constitutional_triad.md` | What three principles govern the framework's operation? |
| `cil_000_scope_boundary.md` | What does Admissibility Theory explicitly *not* do? |

If someone asks *"What is Admissibility Theory?"* — these three documents should be sufficient to answer it.

The constitution is stable. Changes to this layer require strong justification. Subsequent layers (doctrine, formalism, examples, simulations) must remain consistent with it.
