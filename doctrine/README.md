# Doctrine

The doctrine layer translates the constitutional framework into operational tools.

Where the constitution answers *"what conditions must a claim satisfy?"*, the doctrine answers *"how do we evaluate claims in practice?"*

| Document | Purpose |
|---|---|
| `maturity_spectrum.md` | Classifies where any research program currently sits between proposal and demonstration |
| `one_page_doctrine.md` | The six-question admission test — the operational form of the Four Axioms |

The doctrine layer is stable but not constitutional. Updates are permitted as the framework encounters new classes of claim.
