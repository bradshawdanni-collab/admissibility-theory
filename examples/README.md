# Examples

The examples layer applies Admissibility Theory to concrete cases.

Frameworks are understood through worked examples faster than through axioms. This layer is the bridge between the constitutional structure and practical use.

Each example applies the Six Questions from `doctrine/one_page_doctrine.md` and locates the claim on the Maturity Spectrum from `doctrine/maturity_spectrum.md`.

The output of every assessment is always a *location*, not a verdict on correctness.
